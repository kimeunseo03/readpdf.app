# 등기부등본 PDF 자동 분석 · 담보 안전성 계산기

회사 내부용 등기부등본 판독 및 실거래 기반 담보 안전성 분석 도구입니다. Vercel + GitHub 배포를 전제로 한 Next.js App Router 프로젝트입니다.

## 구현 범위

### PDF 판독
- PDF 업로드 UI (드래그&드롭, 최대 10MB)
- `/api/documents/parse` 백엔드 API
- PDF 파일 검증 및 `pdf-parse` 기반 텍스트 추출
- OCR 필요 여부 판단
- 등본 키워드/구조 탐지
- 주소, 건물명, 동, 호수, 전용면적, 층수, 건축연도 추출
- 선순위 근저당권, 압류, 가압류, 전세권/임차권, 신탁 리스크 플래그 탐지
- 개인정보 마스킹 텍스트 미리보기
- 판독 결과 편집 모드 (오류 항목 직접 수정 가능)

### 실거래 조회
- `/api/transactions/raw` — 국토부 공공데이터 아파트매매 실거래가 다단계 조회
- 카카오 로컬 API를 통한 주소 → 법정동코드 자동 변환
- 6단계 확장 조회 전략 (기간·면적·층수 오차 단계별 확대, 충분한 건수 확보 시 조기 종료)
- 동일 단지 우선 조회 → 부족 시 동일 읍면동 유사단지 폴백
- 해제(취소) 거래 자동 제외, 중복 거래 제거

### 담보 안전성 계산
- `/api/valuation` — 아파트 종합 감정 평가
- 선택 거래 기반 IQR 이상치 제거 및 보정 평균가 산출
- 면적 차이 가중 평균가 계산
- 거래별 추천·주의·참고 태그 자동 부여
- 선순위 근저당 + 임차보증금(전세/월세 환산) 차감 후 담보여력 계산
- 월세 → 전세 환산: `보증금 + (월세 × 12 / 전월세전환율 5.5%)`

## 디렉터리 구조

```txt
app/                              # Next.js 라우트와 API 엔드포인트
  api/
    documents/parse/route.ts      # PDF 판독 API
    transactions/raw/route.ts     # 실거래 조회 API
    valuation/route.ts            # 감정 평가 API
  upload/page.tsx
src/
  backend/                        # 서버 전용 로직
    pdf/                          # PDF 검증, 텍스트 추출, 파싱
    compliance/                   # PII 마스킹, 데이터 소스 정책
    valuation/                    # 가치평가, 실거래 API, 주소 검색, 단지명 정규화
  frontend/                       # 클라이언트 UI 컴포넌트
    components/
      UploadForm.tsx               # PDF 업로드 진입점, 스텝바
      ParseResultView.tsx          # 판독 결과 + 조회 조건 + 상태 관리
      RawTransactionLookup.tsx     # 실거래 테이블 (순수 표시 컴포넌트)
      ValuationForm.tsx            # 감정 평가 폼
    lib/
  shared/                         # 프론트/백엔드 공용 타입
docs/                             # 설계 가이드라인
```

## 환경 변수

```env
PUBLIC_DATA_API_KEY=   # 국토부 공공데이터포털 아파트매매 실거래가 API 키
KAKAO_REST_API_KEY=    # 카카오 로컬 주소 검색 API 키
```

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000/upload` 접속 후 PDF를 업로드합니다.

## Vercel 배포

1. GitHub에 이  프로젝트를 push합니다.
2. Vercel에서 해당 repository를 Import합니다.
3. Framework Preset은 Next.js로 자동 인식됩니다.
4. Environment Variables에 `PUBLIC_DATA_API_KEY`, `KAKAO_REST_API_KEY`를 등록합니다.

## 현재 한계

- OCR 실제 수행은 구현하지 않았고 필요 여부만 판단합니다.
- 스캔 이미지 기반 PDF는 텍스트 추출 불가입니다.
- 등본 서식마다 표현이 다를 수 있어 정규식 추출 정확도는 검증 데이터로 개선해야 합니다.
- 아파트 외 물건(빌라·오피스텔·단독주택)은 실거래 데이터 부족으로 신뢰도 낮습니다.
- 전월세전환율 5.5% 고정 적용으로 실제 전환율과 차이가 발생할 수 있습니다.
- 원본 PDF 저장, DB 저장, 수정 이력 저장은 포함하지 않았습니다.
- 본 시스템의 분석 결과는 참고용이며 법적 효력 있는 감정평가를 대체하지 않습니다.
